# Coercion Techniques

## Overview

Many NTLM Relay scenarios rely on coercing systems into authenticating toward attacker-controlled services.

---

# Common Coercion Sources

* Printer services
* File services
* Remote management protocols
* Legacy authentication mechanisms

---

# Assessment Focus

Evaluate:

* Authentication exposure
* Service trust relationships
* Administrative systems
* Tier 0 assets

---

# Enterprise Risks

* Forced authentication
* Credential relay
* Privilege escalation

---

# Defensive Considerations

* Restrict unnecessary services
* Harden authentication protocols
* Monitor unusual authentication behavior
