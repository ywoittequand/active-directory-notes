# Relay Scenarios

## Overview

NTLM Relay attacks typically involve capturing authentication attempts and relaying them to another service.

---

# Common Relay Targets

## SMB Relay

Potential Impact:

* Administrative access
* Remote operations

---

## LDAP Relay

Potential Impact:

* Object modification
* ACL abuse
* Privilege escalation

---

## HTTP Relay

Potential Impact:

* Web application abuse
* Administrative compromise

---

# Assessment Methodology

1. Identify NTLM-enabled services
2. Review signing protections
3. Analyze relay opportunities
4. Evaluate privilege impact

---

# Enterprise Risks

* Attack path expansion
* Administrative compromise
* Tier 0 exposure
