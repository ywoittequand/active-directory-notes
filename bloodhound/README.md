# BloodHound Attack Path Analysis

## Overview

BloodHound is a graph-based Active Directory analysis platform used to identify privilege escalation paths, hidden permissions, and lateral movement opportunities.

## Assessment Objectives

- Identify attack paths
- Detect privilege escalation opportunities
- Analyze ACL abuse
- Review delegation exposure
- Evaluate lateral movement possibilities

## Common Findings

- Excessive permissions
- Local admin chaining
- Delegation misconfigurations
- Session exposure
- Weak tiering

## Frequently Used Tools

- BloodHound
- SharpHound
- Neo4j

## Defensive Considerations

- Tiered administration
- Session isolation
- Least privilege
- ACL reviews
